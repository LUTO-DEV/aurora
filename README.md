# 🌌 AURORA | AI Design Partner

> "Most AI tools generate visuals. Aurora generates design thinking first."

AURORA is a strategic design partner that bridges the gap between raw ideas and professional UI/UX. Instead of jumping straight to pixels, it analyzes audience psychology, establishes a design system, and generates interactive, context-aware mockups.



## ✨ Core Features

- **🧠 Strategic Reasoning:** Generates full design briefs including audience psychology, typography pairing, and color theory justification.
- **🎨 Dynamic Mockups:** Real-time UI rendering with 9+ layout types (E-commerce, SaaS, Social, Media, etc.).
- **📱 Auto-Responsive:** Intelligent device switching with dedicated mobile frames and multi-viewport previews.
- **🛠 Developer Ready:** Export systems for Tailwind CSS, SCSS, JSON tokens, and professional PDF design briefs.
- **⚡ High Performance:** Built with Groq (Llama 3.3) for near-instant reasoning.

## 🛠 Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Zustand (State Management)
- **Backend:** Node.js, Express
- **AI:** Groq API (Llama 3.1 / 3.3)
- **Exports:** jsPDF, Lucide Icons
